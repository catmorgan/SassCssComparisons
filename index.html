Sass and CSS Comparisons

Nesting (Parent-Child Relationship)

Sass

This is a child div styled with sass. 
                    .sass-container {
                        .child {
                            color: red;
                        }
                    }

                    .parent {
                        .child {
                          border: thin solid black;
                        }
                    }

CSS

This is a child div styled with css. 
                    .css-container .child {
                        color: blue;
                    }
                    
                    .parent .child {
                        background-color: yellow;
                    }
