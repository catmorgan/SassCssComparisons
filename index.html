Sass and CSS Comparisons

GitHub Repository

Nested Rules

Sass

This is a child div styled with sass. 
                    .sass-container {
                        .child {
                            color: red;
                        }
                    }

CSS

This is a child div styled with css. 
                    .css-container .child {
                        color: blue;
                    }
This is a child div styled with sass, and the css styles are also being applied to it. 
                    .parent {
                        .child {
                            border: thin solid black;
                        }
                    }
This is a child div styled with css, and the scss styles are also being applied to it. 
                    .parent .child {
                        background-color: yellow;
                    }

Parent Selectors: &

Sass

Hover over me! I am styled with sass. 
                    .sass-container {
                        .hover {
                            &:hover {
                            text-decoration: none;
                            background-color: purple;
                            color: white;
                            }
                        }
                    }

CSS

Hover over me! I am styled with css. 
                    .css-container .hover:hover {
                        background-color: blue;
                        color: green;
                        text-decoration: underline;
                    }
